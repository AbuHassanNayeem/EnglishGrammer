## üéØ Executive Summary

### What is Conventional Implicature?
**Conventional implicature** refers to meaning that is linguistically encoded (built into specific words/constructions) but **does not affect truth conditions**. Unlike conversational implicature, it is:
- ‚úÖ **Non-cancellable** (can't be denied without contradiction)
- ‚úÖ **Context-independent** (meaning is lexically fixed)
- ‚úÖ **Truth-preserving** (doesn't change sentence truth value)

## üìù Study Tips & Strategies

### Active Learning Techniques

#### 1. **Example Creation Method**
- Write 5 original sentences for each particle type
- Vary contexts (formal/informal, written/spoken)
- Exchange with peers for feedback

#### 2. **Error Detection Practice**
- Find particle misuse in sample texts
- Identify formality mismatches
- Correct punctuation errors with discourse connectives

#### 3. **Substitution Exercises**
- Replace particles while maintaining meaning
- Example: "He's smart. Moreover, he's diligent." ‚Üí "He's smart. Furthermore, he's diligent."
- Note: Some substitutions change register!

#### 4. **Cancellation Test**
- Try to cancel the implicature (should fail!)
- Example: "John is poor but happy. #But I don't mean to contrast." ‚ùå

#### 5. **Formality Matching**
- Rewrite informal sentences formally
- Example: "It rained, so the game was off." ‚Üí "It rained. Consequently, the match was canceled."

### Mnemonic Devices

#### **COFFEE Types** (6 Core Categories)
- **C**ontrastive (but, yet)
- **O**nly/Focus (only, even, just)
- **F**urthermore/Additive (and, also, moreover)
- **F**ortunately/Evaluative (luckily, unfortunately)
- **E**ven-so/Concessive (still, nevertheless)
- **E**rgo/Conclusive (therefore, thus)

#### **Non-Cancellability Principle**
> "Once triggered, can't be untriggered"

#### **Truth-Preserving Nature**
> "Implicature adds layer, doesn't change core"

#### **Position Sensitivity** (Focus Particles)
> "**Only** placement = meaning displacement"
- "Only JOHN passed" (others didn't)
- "John only PASSED" (didn't excel)
- "John passed ONLY" (and nothing else)

### Common Mistake Prevention

| ‚úÖ DO | ‚ùå DON'T |
|:------|:---------|
| Place focus particles immediately before focused element | Use "also" and "too" together (redundant) |
| Use semicolon/period before "therefore," "nevertheless" | Create comma splices with conclusive particles |
| Match particle formality to context | Use "so" or "plus" in academic essays |
| Check presuppositions with additive particles | Assume "but" and "and" are interchangeable |
| Vary particles to demonstrate vocabulary range | Repeat the same particle excessively |
