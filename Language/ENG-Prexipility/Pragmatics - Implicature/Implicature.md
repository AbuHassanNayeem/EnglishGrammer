**IMPLICATURE**
├── **Definition**
│   ├── Meaning **suggested or implied** by a speaker, **not explicitly stated**, but **inferred** by the listener using context, logic, and shared knowledge.
│   └── Coined by **H. P. Grice (1975)** under **Pragmatics** (meaning in use, not form).
│
├── **Core Function**
│   ├── Enables **indirect communication**
│   ├── Adds **economy, politeness, irony, sarcasm**
│   ├── Allows speakers to convey **more meaning with fewer words**
│   └── Tests **pragmatic competence** in competitive exams
│
├── **Theoretical Foundation**
│   ├── **Grice’s Cooperative Principle (CP)**
│   │   ├── Speakers normally cooperate to communicate effectively
│   │   └── Implicature arises when CP is **observed, flouted, or violated**
│   │
│   └── **Maxims of Conversation**
│       ├── Quantity
│       ├── Quality
│       ├── Relation (Relevance)
│       └── Manner
│
├── **Main Types**
│   ├── **Conversational Implicature**
│   │   ├── **Definition**
│   │   │   └── Implicature derived from **context + Gricean maxims**, not fixed meaning
│   │   │   ├── A meaning inferred by the listener/reader from the context of a conversation, 
│   │   │   │   rather than explicitly stated by the speaker.
│   │   │   ├── Proposed by H.P. Grice in his theory of implicature (1975).
│   │   │   └── Relies on the **Cooperative Principle**: "Make your contribution such as is required, at the stage at which it occurs, by the accepted purpose or direction of the talk exchange."
│   │   ├── **Key Properties**
│   │   │   ├── Cancellable
│   │   │   ├── Context-dependent
│   │   │   ├── Non-conventional
│   │   │   └── Calculable via reasoning
│   │   │   ├── **Cancellable Implicature**
│   │   │   │   ├── **Definition**: Can be explicitly cancelled or negated without contradiction.
│   │   │   │   ├── **Function**: Shows that implicature is not part of literal meaning.
│   │   │   │   ├── **Examples**
│   │   │   │   │   ├── *John has three children; in fact, he may have more.*  
│   │   │   │   │   └── *Some of the students passed the exam, maybe all of them did.*  
│   │   │   │   ├── **Variations**
│   │   │   │   │   ├── Explicit cancellation: *…but not necessarily…*  
│   │   │   │   │   └── Contextual cancellation: changes with setting.  
│   │   │   │   └── **Exam Tip**
│   │   │   │       └── “If a meaning can be cancelled, it’s implicature, not entailment.”
│   │   │   │
│   │   │   ├── **Context-dependent Implicature**
│   │   │   │   ├── **Definition**: Depends on situational, cultural, or linguistic context to be understood.
│   │   │   │   ├── **Function**: Shows importance of speaker-listener shared knowledge.
│   │   │   │   ├── **Examples**
│   │   │   │   │   ├── *Can you pass the salt?* → request, not literal question.
│   │   │   │   │   └── *It’s cold in here.* → may imply “Please close the window.”
│   │   │   │   ├── **Transformations**
│   │   │   │   │   └── Same sentence may carry different implicature in different settings.
│   │   │   │   └── **Exam Tip**
│   │   │   │       └── “Always check context before marking as implicature in MCQs.”
│   │   │   │
│   │   │   ├── **Non-conventional Implicature**
│   │   │   │   ├── **Definition**: Not fixed by words or grammar; arises from conversational reasoning.
│   │   │   │   ├── **Function**: Differentiates from conventional meanings like idioms.
│   │   │   │   ├── **Examples**
│   │   │   │   │   ├── *He is poor but happy.* → “but” implies contrast beyond literal conjunction.
│   │   │   │   │   └── *She is young and talented.* → implied admiration.  
│   │   │   │   ├── **Transformations**
│   │   │   │   │   └── Can often be rephrased with explicit reasoning: *He is poor, however, he is happy.*
│   │   │   │   └── **Exam Tip**
│   │   │   │       └── “Non-conventional implicature is always context-sensitive; watch for conjunctions, connectors.”
│   │   │   │
│   │   │   ├── **Calculable Implicature**
│   │   │   │   ├── **Definition**: Inferred through reasoning using conversational maxims (Quantity, Quality, Relation, Manner).
│   │   │   │   ├── **Function**: Allows systematic derivation of implied meaning.
│   │   │   │   ├── **Examples**
│   │   │   │   │   ├── *Some of the students passed.* → Implies *Not all passed* (Quantity maxim).  
│   │   │   │   │   └── *He went to the party, and he drank a lot.* → Implicates carelessness (Manner/Relation).  
│   │   │   │   ├── **Variations**
│   │   │   │   │   ├── Based on single maxim violation.
│   │   │   │   │   └── Based on combination of multiple maxims.
│   │   │   │   └── **Exam Tip**
│   │   │   │       └── “Check which Gricean maxim is flouted to derive calculable implicature.”
│   │   │   │
│   │   │   ├── **Advanced / Edge Cases**
│   │   │   │   ├── Implicature vs Presupposition: Presupposition survives negation, implicature may not.
│   │   │   │   ├── Flouted Maxims: Sometimes speakers deliberately flout maxims to imply humor, irony, or politeness.
│   │   │   │   ├── Cultural/Regional Variations: Same sentence may yield different implicature across cultures.
│   │   │   │   └── Embedded Implicatures: Implicatures within subordinate clauses or complex sentences.
│   │   │   │
│   │   │   ├── **Common Mistakes**
│   │   │   │   ├── Confusing implicature with entailment (entailment cannot be cancelled).
│   │   │   │   ├── Ignoring context, leading to misinterpretation.
│   │   │   │   ├── Over-attributing implicature where literal meaning suffices.
│   │   │   │   └── Missing flouting of maxims as a clue for implicature.
│   │   │   │
│   │   │   ├── **Comparison with Similar Structures**
│   │   │   │   ├── **Entailment**
│   │   │   │   │   ├── Always true if the main statement is true.
│   │   │   │   │   └── Cannot be cancelled.  
│   │   │   │   ├── **Presupposition**
│   │   │   │   │   ├── Background assumption, survives negation.
│   │   │   │   │   └── e.g., *The king of France is bald.* (assumes existence).  
│   │   │   │   └── **Implication in Semantics**
│   │   │   │       ├── Conventional vs conversational: conventional is lexical, conversational is inferred.
│   │   │   │       └── Implicature is cancellable, context-sensitive, and reasoning-based.
│   │   │   │
│   │   │   └── **Quick Revision Rules**
│   │   │       ├── “Cancellable = check negation”  
│   │   │       ├── “Context-dependent = always consider setting”  
│   │   │       ├── “Non-conventional = not fixed by words”  
│   │   │       ├── “Calculable = derive via Gricean maxims”  
│   │   │       ├── “Implicature ≠ Entailment ≠ Presupposition”  
│   │   │       ├── “Flouting maxims often signals indirect meaning”  
│   │   │       └── “Exam Tip: Look for indirectness, optionality, reasoning cues, and contrastive connectors.”
│   │   │   
│   │   ├── **Core Function**
│   │   │   ├── To convey additional meaning indirectly.
│   │   │   ├── To enrich communication without stating everything explicitly.
│   │   │   ├── To allow economical, flexible language use.
│   │   │   └── To create politeness, irony, humor, or subtlety in speech.
│   │   ├── **Subtypes**
│   │   │   ├── **Generalized Conversational Implicature (GCI)**
│   │   │   │   ├── **Definition**
│   │   │   │   │   └── Inference normally drawn **without special context**
│   │   │   │   │
│   │   │   │   ├── **Function**
│   │   │   │   │   └── Adds default assumptions in ordinary usage
│   │   │   │   │
│   │   │   │   ├── **Examples**
│   │   │   │   │   ├── *Some students passed the exam.*
│   │   │   │   │   │   → Implicature: *Not all students passed.*
│   │   │   │   │   └── *I met a woman yesterday.*
│   │   │   │   │       → Implicature: *She was not the speaker’s wife.*
│   │   │   │   │
│   │   │   │   ├── **Variations / Forms**
│   │   │   │   │   ├── Scalar terms: some, many, often
│   │   │   │   │   └── Indefinites: a, an
│   │   │   │   │
│   │   │   │   ├── **Transformations**
│   │   │   │   │   └── Cancellable: *Some students passed — in fact, all did.*
│   │   │   │   │
│   │   │   │   └── **Competitive Exam Tips**
│   │   │   │       └── “**Some ≠ all** unless explicitly stated”
│   │   │   │
│   │   │   ├── **Particularized Conversational Implicature (PCI)**
│   │   │   │   ├── **Definition**
│   │   │   │   │   └── Inference depends on **specific situational context**
│   │   │   │   │
│   │   │   │   ├── **Function**
│   │   │   │   │   └── Conveys indirect meaning using shared situational knowledge
│   │   │   │   │
│   │   │   │   ├── **Examples**
│   │   │   │   │   ├── *A: Are you going to the party?*
│   │   │   │   │   └── *B: I have an exam tomorrow.*
│   │   │   │   │       → Implicature: *No, I’m not going.*
│   │   │   │   │
│   │   │   │   ├── **Transformations**
│   │   │   │   │   └── Cannot be inferred without context
│   │   │   │   │
│   │   │   │   └── **Competitive Exam Tips**
│   │   │   │       └── Context clues decide meaning — **MCQ favorite**
│   │   │   │
│   │   │   ├── **Scalar Implicature**
│   │   │   │   ├── **Definition**
│   │   │   │   │   └── Meaning inferred from **choice on an information scale**
│   │   │   │   │
│   │   │   │   ├── **Common Scales**
│   │   │   │   │   ├── all > most > many > some
│   │   │   │   │   ├── always > often > sometimes
│   │   │   │   │   └── must > should > may
│   │   │   │   │
│   │   │   │   ├── **Examples**
│   │   │   │   │   └── *She is sometimes late.*
│   │   │   │   │       → Implicature: *Not always late.*
│   │   │   │   │
│   │   │   │   └── **Exam Tip**
│   │   │   │       └── Weaker term implies negation of stronger
│   │   │   │
│   │   │   └── **Maxim-based Implicature**
│   │   │       ├── **Quantity-based**
│   │   │       │   ├── *He has three children.*
│   │   │       │   └── Implicature: *Exactly three*
│   │   │       ├── **Quality-based**
│   │   │       │   ├── *He is a real Einstein.*
│   │   │       │   └── Irony / sarcasm
│   │   │       ├── **Relation-based**
│   │   │       │   └── Answer seems irrelevant but is meaningful
│   │   │       └── **Manner-based**
│   │   │           └── Unusual phrasing implies marked meaning
│   │   │
│   │   ├── **Restrictions & Exceptions**
│   │   │   ├── Not truth-conditional
│   │   │   └── Can be cancelled explicitly
│   │   │
│   │   └── **Common MCQ Traps**
│   │       ├── Confusing implicature with literal meaning
│   │       └── Ignoring context
│   │
│   ├── **Conventional Implicature**
│   │   ├── **Definition**
│   │   │   └── Implicature **fixed by specific words**, not context
│   │   │
│   │   ├── **Key Properties**
│   │   │   ├── Not cancellable
│   │   │   ├── Context-independent
│   │   │   └── Lexically triggered
│   │   │
│   │   ├── **Trigger Words**
│   │   │   ├── but
│   │   │   ├── even
│   │   │   ├── yet
│   │   │   └── therefore
│   │   │
│   │   ├── **Examples**
│   │   │   ├── *He is poor but honest.*
│   │   │   │   → Implicature: *Poverty contrasts with honesty*
│   │   │   └── *Even John passed.*
│   │   │       → Implicature: *John was least expected*
│   │   │
│   │   ├── **Transformations**
│   │   │   └── Cannot be removed without changing meaning
│   │   │
│   │   └── **Competitive Exam Tips**
│   │       └── “**but = contrast** always”
│
├── **Advanced / Edge Cases**
│   ├── **Implicature vs Presupposition**
│   │   ├── Implicature → cancellable
│   │   └── Presupposition → survives negation
│   │
│   ├── **Implicature vs Inference**
│   │   ├── Implicature → speaker-intended
│   │   └── Inference → listener-generated
│   │
│   ├── **Irony & Sarcasm**
│   │   └── Maxim of Quality deliberately flouted
│   │
│   └── **Silence-based Implicature**
│       └── Absence of reply implies meaning
│
├── **Common Mistakes**
│   ├── Treating implicature as grammar rule
│   ├── Ignoring cancellability
│   ├── Mixing literal meaning with implied meaning
│   └── Assuming all implications are implicatures
│
├── **Comparison with Similar Structures**
│   ├── Implicature vs Explicit Meaning
│   ├── Implicature vs Entailment
│   ├── Implicature vs Presupposition
│   └── Pragmatics vs Semantics
│
├── **British vs American Exam Preferences**
│   ├── British exams (IELTS, BCS)
│   │   └── Focus on **pragmatic interpretation**
│   └── American exams (GRE, TOEFL)
│       └── Focus on **speaker intention & context**
│
└── **Quick Revision Rules**
├── Implied ≠ stated
├── Context creates meaning
├── Cancellable → Conversational
├── Word-triggered → Conventional
├── “Some” usually implies “not all”
└── Maxims explain hidden meaning
