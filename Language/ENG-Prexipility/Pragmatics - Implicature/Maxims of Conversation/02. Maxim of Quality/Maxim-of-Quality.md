**MAXIM OF QUALITY (TRUTHFULNESS) — GRICEAN PRAGMATICS**
├── **Definition**
│   ├── A conversational principle stating that speakers should **say what they believe to be true** and **avoid saying what they lack evidence for**.
│   ├── Proposed by **H. P. Grice (1975)** under the **Cooperative Principle**.
│   └── Concerned with **truth and evidence**, not relevance, quantity, or clarity.
│
├── **Core Function**
│   ├── Ensures **reliability and credibility** of communication
│   ├── Allows listeners to **trust assertions** unless signaled otherwise
│   ├── Explains **irony, sarcasm, exaggeration, metaphor** via implicature
│   └── Central to **dialogue interpretation questions** in exams
│
├── **Main Types**
│   ├── **Observance of Maxim of Quality**
│   │   ├── **Definition**
│   │   │   └── Speaker provides information that is **true and well-supported by evidence**.
│   │   │
│   │   ├── **Function / Role**
│   │   │   └── Produces **literal meaning** without implicature.
│   │   │
│   │   ├── **Examples**
│   │   │   ├── *Water boils at 100°C.*
│   │   │   └── *He arrived at 9 a.m. as scheduled.*
│   │   │
│   │   ├── **Variations / Forms**
│   │   │   ├── Statements of fact
│   │   │   ├── Verified reports
│   │   │   └── Evidence-based claims
│   │   │
│   │   ├── **Transformations**
│   │   │   └── Opinion → fact-based assertion (with evidence)
│   │   │
│   │   ├── **Restrictions & Exceptions**
│   │   │   └── Absolute truth not required; **reasonable belief + evidence** sufficient
│   │   │
│   │   └── **Competitive Exam Tips**
│   │       └── Literal, factual answers usually indicate **observance**, not implicature
│   │
│   ├── **Flouting of Maxim of Quality**
│   │   ├── **Definition**
│   │   │   └── Speaker **deliberately says something untrue or exaggerated** to create implicature.
│   │   │
│   │   ├── **Function / Role**
│   │   │   └── Generates **non-literal meaning** (irony, sarcasm, metaphor).
│   │   │
│   │   ├── **Subtypes**
│   │   │   ├── **Irony**
│   │   │   │   ├── **Definition**: Saying the opposite of what is meant.
│   │   │   │   ├── **Examples**
│   │   │   │   │   └── *What a lovely weather!* (during a storm)
│   │   │   │   ├── **Transformations**
│   │   │   │   │   └── Ironic praise → implicit criticism
│   │   │   │   └── **Exam Tip**
│   │   │   │       └── Tone + context override literal meaning
│   │   │   │
│   │   │   ├── **Sarcasm**
│   │   │   │   ├── **Definition**: Sharp, often mocking irony.
│   │   │   │   ├── **Examples**
│   │   │   │   │   └── *Great job!* (after a mistake)
│   │   │   │   ├── **Restrictions**
│   │   │   │   │   └── Requires shared social awareness
│   │   │   │   └── **Exam Tip**
│   │   │   │       └── Sarcasm = flouting Quality, not violating
│   │   │   │
│   │   │   ├── **Hyperbole (Exaggeration)**
│   │   │   │   ├── **Definition**: Deliberate overstatement for effect.
│   │   │   │   ├── **Examples**
│   │   │   │   │   └── *I’ve told you a thousand times.*
│   │   │   │   ├── **Transformations**
│   │   │   │   │   └── Exaggeration → realistic statement
│   │   │   │   └── **Exam Tip**
│   │   │   │       └── Numbers are common hyperbole markers
│   │   │   │
│   │   │   └── **Metaphor / Figurative Speech**
│   │   │       ├── **Definition**: Non-literal comparison implying meaning.
│   │   │       ├── **Examples**
│   │   │       │   └── *Time is a thief.*
│   │   │       ├── **Transformations**
│   │   │       │   └── Metaphor → literal explanation
│   │   │       └── **Exam Tip**
│   │   │           └── Figurative ≠ false; it signals implicature
│   │   │
│   │   ├── **Variations / Forms**
│   │   │   ├── Overstatement
│   │   │   ├── Understatement
│   │   │   └── Figurative language
│   │   │
│   │   ├── **Restrictions & Exceptions**
│   │   │   ├── Listener must detect **intentional untruth**
│   │   │   └── Context essential for inference
│   │   │
│   │   └── **Competitive Exam Tips**
│   │       ├── Ask: *Is the speaker obviously not being literal?*
│   │       └── If yes → flouting Quality
│   │
│   ├── **Violation of Maxim of Quality**
│   │   ├── **Definition**
│   │   │   └── Speaker **lies or provides false information** intending to deceive.
│   │   │
│   │   ├── **Function / Role**
│   │   │   └── Breaks cooperation; **no implicature** intended.
│   │   │
│   │   ├── **Examples**
│   │   │   └── *I was at home all day.* (when speaker knows it is false)
│   │   │
│   │   ├── **Restrictions**
│   │   │   └── Listener unaware of deception at the time
│   │   │
│   │   └── **Exam Tip**
│   │       └── Deception without contextual signal = violation
│   │
│   └── **Suspension of Maxim of Quality**
│       ├── **Definition**
│       │   └── Truthfulness temporarily irrelevant due to **genre or convention**.
│       │
│       ├── **Contexts**
│       │   ├── Jokes
│       │   ├── Fiction
│       │   ├── Fairy tales
│       │   └── Hypothetical scenarios
│       │
│       └── **Exam Tip**
│           └── Rarely tested; avoid confusing with flouting
│
├── **Advanced / Edge Cases**
│   ├── **Quality vs Belief**
│   │   └── Speaker must believe statement to be true, not guarantee absolute truth
│   │
│   ├── **Hedging Devices**
│   │   ├── *I think, probably, it seems*
│   │   └── Used to **protect Quality** by lowering commitment
│   │
│   ├── **Multiple Maxims Interaction**
│   │   └── Quality flouted alongside Manner or Quantity
│   │
│   └── **Formal vs Informal Usage**
│       ├── Formal (academic, legal): strict Quality
│       └── Informal (conversation): flexible, figurative
│
├── **Common Mistakes**
│   ├── Treating irony as lying
│   ├── Confusing exaggeration with violation
│   ├── Ignoring context and tone
│   └── Assuming all false statements break cooperation
│
├── **Comparison with Similar Structures**
│   ├── Quality vs Quantity
│   │   └── Quality = truth | Quantity = amount
│   ├── Quality vs Manner
│   │   └── Quality = truthfulness | Manner = clarity
│   ├── Implicature vs Literal Falsehood
│   │   └── Implicature intended, falsehood concealed
│   └── Pragmatics vs Semantics
│
├── **British vs American Exam Preferences**
│   ├── **British Exams (IELTS, BCS, SSC)**
│   │   └── Emphasize **irony, understatement, politeness strategies**
│   └── **American Exams (TOEFL, GRE)**
│       └── Emphasize **logical detection of exaggeration and sarcasm**
│
└── **Quick Revision Rules**
├── Quality = **be truthful**
├── Obvious untruth → check implicature
├── Irony = flouting, not violation
├── Lying = violation
├── Figurative language signals meaning beyond truth
├── Context decides interpretation
└── Memory Hack: **Q = Quality = Question truth**
