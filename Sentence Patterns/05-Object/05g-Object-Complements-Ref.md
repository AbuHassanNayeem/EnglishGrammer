## 1. Definition
An **Object Complement (Co)** follows the Direct Object and renames or describes it.
*   *Structure:* Verb + Object + Object Complement.
*   *Logic:* It answers "What?" or "How?" the object became/is.

---

## 2. Types of Object Complements

| Type | Example | Explanation |
| :--- | :--- | :--- |
| **Noun** | *They elected him **president**.* | Renames `Him` -> `President`. |
| **Adjective** | *The joke made me **angry**.* | Describes `Me` -> `Angry`. |
| **Infinitive** | *I want you **to study**.* | Expected action of `You`. |
| **Gerund** | *We found him **sleeping**.* | Ongoing action of `Him`. |

---

## 3. Difference from Subject Complement
*   **Subject Complement:** Describes the Subject. (*He is angry*).
*   **Object Complement:** Describes the Object. (*The joke made **him** angry*).

---

## 4. Common Verbs
Verbs that take object complements: *make, call, name, elect, find, consider, appoint, keep, choose, declare.*
