## 1. The General Rule
Phrasal verbs are extremely common in **Spoken English** and **Informal Writing**.
In **Academic** or **Formal Business Writing**, single-word Latinate equivalents are often preferred.

*   *Informal:* The problem **went away**.
*   *Formal:* The problem **disappeared**.

---

## 2. Comparison Table

| Phrasal Verb | Formal Equivalent | Sentence (Formal) |
| :--- | :--- | :--- |
| **Bring about** | Cause | This **caused** a change. |
| **Call off** | Cancel | The event was **cancelled**. |
| **Find out** | Discover/Ascertain | We must **ascertain** the truth. |
| **Get away** | Escape | The suspect **escaped**. |
| **Give in** | Yield/Submit | They refused to **submit**. |
| **Go up** | Increase | Prices **increased**. |
| **Go down** | Decrease | Sales **decreased**. |
| **Leave out** | Omit | Please do not **omit** details. |
| **Look at** | Examine | Please **examine** the data. |
| **Look into** | Investigate | We will **investigate** the issue. |
| **Make up** | Fabricate | He **fabricated** the story. |
| **Point out** | Indicate | Results **indicate** a trend. |
| **Put off** | Postpone | The meeting is **postponed**. |
| **Put up with** | Tolerate | We cannot **tolerate** this behavior. |
| **Set up** | Establish | The firm was **established** in 1990. |
| **Talk about** | Discuss | We will **discuss** the matter. |
| **Think about** | Consider | Please **consider** the offer. |
| **Throw away** | Discard | Do not **discard** waste here. |

---

## 3. When to Use Which?
*   **Email to friend:** *Can we **put off** the meeting?* (Phrasal Verb OK).
*   **Report to CEO:** *It is recommended that the meeting be **postponed**.* (Formal preferred).
*   **Newspaper Headline:** *Government **Sets Up** New Committee.* (Phrasal Verbs used for brevity/punchiness).

## 4. Practice Exercises

### Convert to Formal:
1. We need to **look into** the problem.
2. They **turned down** the offer.

> **Answers:**
> 1. We need to **investigate** the problem.
> 2. They **rejected** the offer.
